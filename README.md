# Charge Capture Prototype

A React prototype for a desktop feature flow with a drawer component for adding text snippets.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Features

- Right-side drawer that slides in from the edge
- Tab slider UI to switch between Basic and Advanced text snippet views
- Clean, modern UI matching the Figma design
